### 폴더 구조

```bash
src
┣ app : Next.js App Router
┃ ┣ (pages) : 페이지
┃ ┣ api : API 라우트
┃ ┣ globals.css
┃ ┣ layout.tsx
┃ ┗ Providers.tsx
┣ components : 공용 컴포넌트
┣ constants : 상수
┣ fonts : 폰트 파일
┣ hooks : 커스텀 훅 모음
┣ lib : 공용 유틸/라이브러리
┣ service : 비즈니스 로직 서비스 계층
┣ types : 전역 타입 정의
┣ utils : 유틸리티 함수
┣ zustand : 전역 상태 관리
┗ middleware.ts : 미들웨어 설정
```
